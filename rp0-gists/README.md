# RP-0 Gists

Short instructor-facing gist summaries for RP-0 research proposal pitches.

Files in this folder:
- `class-overview.md` — class-wide trends, strongest patterns, and common coaching points
- one file per readable student submission (using `last-first.md` naming where student names are known)
- `missing-submissions-note.md` — notes on submissions that were not readable from the provided attachments

These are meant to be quick reference notes, not final LMS feedback.
